# Mini Fantasy Asset Index - Electron App

This is an experimental Electron App to allow people to search through all the Mini Fantasy assets, offline.

## LLM Disclaimer
This project was also a chance for me to experiment with heavy LLM usage in coding, the bulk of this project code was generated by Claude 3.7 Sonnet with my prompting, supervision, and at many times patchwork and gluing to make sure it all worked in the end.
